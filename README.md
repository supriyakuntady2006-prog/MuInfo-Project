# MuInfo — PWA museum companion (scaffold)

This is a starter scaffold for MuInfo, a Progressive Web App (PWA) that will let visitors scan artworks and view information offline.

Quick start (Windows PowerShell):

```powershell
npm install
npm run dev
```

What’s included

- React + Vite app scaffold
- Basic Camera component (uses getUserMedia)
- PWA manifest and simple service worker placeholder
